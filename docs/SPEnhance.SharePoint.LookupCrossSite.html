SPEnhance.SharePoint.LookupCrossSite

Features

  1. Lookup cross site
  2. Filter data using target list view
  3. Filter data using CAML query

Installation

Run these commands using SharePoint Powershell. Its a farm solution.

  1. add-spsolution -literalpath path-to-wsp
  2. install-spsolution -identity wsp-name -gacdeployment

Example:-

  1. add-spsolution -literalpath C:\temp\SPEnhance.SharePoint.LookupCrossSite.wsp
  2. install-spsolution -identity SPEnhance.SharePoint.LookupCrossSite.wsp -gacdeployment
